## 🔨 Build Setup

1. Clone the repository

```bash
git clone https://github.com/mpont91/cv.git
```

2. Change the working directory

```bash
cd cv
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm run dev
```

🌟 You are all set!

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
